ReconX is a **modular, GUI-driven reconnaissance framework** designed for **authorized bug bounty programs, penetration tests, and security research**.  
It consolidates essential reconnaissance techniques into a unified, professional interface to streamline early-stage security assessments.

## Overview

Reconnaissance is a foundational phase of security testing. ReconX provides a centralized platform for domain enumeration, service discovery, and metadata collection, reducing tooling overhead while improving workflow efficiency.

The framework emphasizes:
- Modular execution
- Clear output and logging
- Ethical and authorized use
- Extensibility for future enhancements

---

## Features

### Recon Modules
- Subdomain Enumeration  
- Port Scanning  
- Directory Bruteforcing  
- Technology Fingerprinting  
- DNS Enumeration  
- HTTP Header Analysis  
- Wayback URL Collection  
- WHOIS Lookup  
- Basic Vulnerability Checks  
- Full Recon (automated execution)

---

## Interface

- Target scope input with validation
- Selective module execution
- Live console output with timestamps
- Structured results and statistics views
- Exportable findings
- Execution controls (Start / Stop / Clear)

---

## Installation

### Requirements
- Python **3.9+**
- Linux environment (Kali Linux recommended)

### Setup
```bash
git clone https://github.com/cipher1004/ReconX.git
cd ReconX
pip install -r requirements.txt
python reconx.py

```
